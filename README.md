# Atividade Prática – Matriz de Episódios com OMDb API

## Objetivo da atividade

Este projeto consiste no desenvolvimento de uma aplicação web que consome a API do OMDb para exibir informações de uma série de TV.

O objetivo principal é mostrar os episódios organizados em forma de matriz, onde cada célula apresenta a classificação (rating) de um episódio. Além disso, a aplicação exibe dados gerais da série, como nome, pôster, ano, gênero, idioma e resumo.

## Instruções de como usar

1- Abra o arquivo index.html no navegador

2- A aplicação irá buscar automaticamente os dados da série na API

3- As informações da série serão exibidas no lado esquerdo

4- A matriz será gerada automaticamente:

- Linhas representam as temporadas
- Colunas representam os episódios

5- Cada célula mostra a nota do episódio com cores diferentes conforme a classificação
